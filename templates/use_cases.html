Coinballer Use Cases

This game is useful in situations where there are multiple parties and participants want to allocate funds to only a single party. In this situation, the fund-raising process double as an election process.

For instance, participants want to allocate funds to either Bitcoin Core or Bitcoin Cash developers, but only to the project that has the most momentum.

In this instance, each party could create a coinballer game, and doners could vote by sending money to the pot via the address for their preferred party.

In another example, governments could hold an election using coinballer, collecting taxes via the game rather than through traditional taxation systems.

Of course, in both these examples, voting power of participants is weighted proportional to the wealth of the participants.
